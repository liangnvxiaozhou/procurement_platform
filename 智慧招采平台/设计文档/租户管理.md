# 租户管理
租户管理负责平台顶层的生产关系配置，支持集团化多级组织架构
## 列表字段

支持根据以下字段组合查询

| 字段名称 | 类型         | 示例                         |
| ---- | ---------- | -------------------------- |
| 租户名称 | String     | 国家投资集团有限公司                 |
| 租户代码 | String(18) | 即社会信用代码，91110000100017643K |
| 租户类型 | Array      | 招标人/采购人                    |
| 联系人  | String     | 张三                         |
| 联系电话 | String     | 15711001992                |
| 入驻时间 | DateTime   | 2026-05-14 14:37:21        |

---

## 单个操作
### 编辑详情

#### 基本信息模块

| 字段名称   | 类型         | 必填  | 说明                |
| ------ | ---------- | --- | ----------------- |
| 租户名称   | String     | 是   | 租户名称，一般为单位名称，全局唯一 |
| 租户类型   | Array      | 是   | 支持多选，招标人/采购人、招标代理 |
| 社会信用代码 | String(18) | 是   |                   |
| 租户编号   | String     | 否   | 系统生成，自增           |
| 联系人    | String     | 否   |                   |
| 联系电话   | String     | 否   |                   |
| 备注     | Text       | 否   |                   |
| 附件     | File       | 否   |                   |

#### 网页配置模块

| 字段名称       | 类型   | 必填  | 说明              |
| ---------- | ---- | --- | --------------- |
| 企业管理后台logo | File | 是   | 分辨率为457x28的图片文件 |
| 采购系统logo   | File | 是   | 分辨率为457x28的图片文件 |
| 监督系统logo   | File | 是   | 分辨率为457x28的图片文件 |

---

### 设置权限

| 字段名称   | 类型      | 必填  | 说明                           |
| ------ | ------- | --- | ---------------------------- |
| 门户网站   | Boolean | 是   | 使用，不使用                       |
| 采购方式   | File    | 是   | 分辨率为457x28的图片文件              |
| 接口平台登录 | Boolean | 是   | 否，是                          |
| 平台使用费  | Boolean | 是   | 收取，不收取                       |
| 自查报告   | Boolean | 是   | 使用，不使用                       |
| 允许招标代理 | Enum    | 是   | 多选，点击后弹窗显示代理机构的企业名称、企业代码，可搜索 |

---

### 关联收费规则
| 字段名称   | 类型   | 必填  | 说明                              |
| ------ | ---- | --- | ------------------------------- |
| 招标规则编号 | Enum | 是   | 可选择平台使用费规则设置中已配置的招标规则，可使用规则名称搜索 |
| 非招规则编号 | Enum | 是   | 可选择平台使用费规则设置中已配置的非招规则，可使用规则名称搜索 |


---
### 查看运营用户
点击后弹窗显示该租户关联的运营管理员，支持设置有效/无效
#### 列表字段

| 字段名称  | 类型      | 说明                              |
| ----- | ------- | ------------------------------- |
| 姓名    | String  | 可选择平台使用费规则设置中已配置的招标规则，可使用规则名称搜索 |
| 用户名   | String  | 可选择平台使用费规则设置中已配置的非招规则，可使用规则名称搜索 |
| 已分配角色 | Array   | 包含多个角色，例如采购管理人员、平台客服、平台运营、信息审核员 |
| 有效/无效 | Boolean | 有效/无效，可开启/关闭                    |
#### 列表操作
支持搜索、开启/关闭账号权限


---

### 查看组织架构
点击后新页面打开组织架构模块，带入搜索条件，定位至顶级租户
### 删除
校验该租户下是否存在已注册的企业，若存在，弹窗提示“该租户下存在有效企业，无法删除！”。若不存在，则执行删除操作。

## 全局操作

### 批量导入
支持下载excel格式的导入模版，添加租户信息，批量导入租户。模版字段如下表

| 字段名称   | 类型      | 必填  | 说明                           |
| ------ | ------- | --- | ---------------------------- |
| 规则名称   | Boolean | 是   | 使用，不使用                       |
| 规则编号   | File    | 是   | 分辨率为457x28的图片文件              |
| 接口平台登录 | Boolean | 是   | 否，是                          |
| 平台使用费  | Boolean | 是   | 收取，不收取                       |
| 自查报告   | Boolean | 是   | 使用，不使用                       |
| 允许招标代理 | Enum    | 是   | 多选，点击后弹窗显示代理机构的企业名称、企业代码，可搜索 |

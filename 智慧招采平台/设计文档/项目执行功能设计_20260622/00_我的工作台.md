# 我的工作台

> 关联文档：[项目执行总览](README.md)

## 1. 业务流程

### 1.1 数据范围

我的工作台以**标段/包**为粒度展示列表，一行一个标段/包（一个项目含多标段包则多行）。数据范围：

- 当前用户为**项目经理**或**项目成员**的项目；
- 且项目状态 = **已确认**（已确认即可执行，不引入额外执行状态机）；
- 取这些项目下的**所有标段/包**。

> 代理项目经理不在本期范围，见待确认问题 5.2。

### 1.2 「进入项目」定位规则

从工作台标段/包列表行操作「进入项目」：

- 跳转项目详情页，定位到**该标段包当前进度对应步骤页签**（首个未完成节点）；
- 无进度时落到「发布」。

项目详情页（采购视角）见 [00_项目详情页.md](00_项目详情页.md)。

## 2. 数据结构 + 状态值

### 2.1 列表字段定义

| 序号 | 字段 | 类型 | 来源 | 说明 |
| --- | --- | --- | --- | --- |
| 1 | 序号 | 数字 | 系统 | 行号 |
| 2 | 标段/包名称 | 文本 | 标段包 | 取自标段包（定义见立项通用版 3.3.1） |
| 3 | 标段/包编号 | 文本 | 标段包 | 系统生成，规则待定 |
| 4 | 项目名称 | 文本 | 父项目 | 带入 |
| 5 | 项目编号 | 文本 | 父项目 | 带入 |
| 6 | 公开方式 | 文本 | 父项目派生 | 公开 / 邀请（派生规则见立项 0608 版 82 行） |
| 7 | 项目状态 | 枚举 | 标段包执行阶段派生 | 见 2.2 |
| 8 | 中标价（万元） | 数字 | 06_定标成交（未设计） | 见 2.2 |
| 9 | 距开标时间 | 计算 | 截标/开标时间或竞价开始时间 | 见 2.2 |
| 10 | 收藏 | 标记 | 用户行为 | 见 2.2 |
| — | 操作 | — | — | 进入项目 |

### 2.2 字段显示规则

**项目状态（标段包执行阶段）**：字段名沿用「项目状态」，语义为该标段包当前所处执行步骤，由各执行子模块进度派生（如招标公告 / 邀请函 / 采购文件 / 标前准备 / 开标 / 评标 / 定标 / 标后…）。具体枚举与派生规则待 02~07 子模块设计完成后补全。

**距开标时间**：取值字段——招标 / 询比 / 谈判 / 直接采购取「截标/开标时间」（公告:145 / 邀请函:165），竞价取「竞价开始时间」（公告:153）；公告 / 邀请函未发布或标段未填时间视为未排期。显示：未排期「—」；未到「Nd/Nh 后」；已过「已开标」。

**中标价（万元）**：成交 / 定标前显「—」；成交 / 定标后显示；竞价以成交价填充。数据来源待 `06_定标成交.md` 设计完成后对接。

**收藏**：每标段包独立，按当前用户持久化。

> 列字段定义见 2.1，显示规则见 2.2，筛选与排序见 3.2。

### 3.2 筛选与排序

**筛选字段**：

| 筛选项 | 类型 | 选项/范围 |
| --- | --- | --- |
| 标段/包名称、标段/包编号、项目名称、项目编号 | 文本搜索 | 模糊匹配 |
| 公开方式 | 下拉单选 | 公开 / 邀请 |
| 项目状态 | 下拉单选 | 枚举待补（见 2.2） |
| 中标价（万元） | 范围 | 最小值 / 最大值 |
| 距开标时间 | 范围 | 如未来 N 天内 |
| 收藏 | 快捷筛选 | 只看收藏 |

> 交易方式筛选条 ≡ 采购方式（6 值）+ 全部；与列表「公开方式」列（公开/邀请派生）不冲突。

**排序**：默认按「距开标时间」升序（最近开标优先），未排期靠后；收藏行在当前排序内置顶。

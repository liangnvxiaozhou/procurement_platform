# 项目立项功能设计（通用版）

> 设计日期：2026-06-12  
> 最后修改：2026-06-12  
> 版本：v1.0

---

## 一、功能概述

采购立项审核通过后，项目经理在「我的项目」页面基于已生效的采购立项建立正式采购项目，划分标段包、关联项目明细、邀请供应商（如适用），为后续招标采购执行提供基础数据。

本文档支持以下 **9 种采购方式**的项目创建：

| 采购方式 | 公开/邀请 |
| ---- | ----- |
| 公开招标 | -     |
| 邀请招标 | -     |
| 询比采购 | 公开    |
| 询比采购 | 邀请    |
| 谈判采购 | 公开    |
| 谈判采购 | 邀请    |
| 竞价   | 公开    |
| 竞价   | 邀请    |
| 直接采购 | -     |

---

## 二、核心业务流程

```
点击新建 → 选择采购立项 → 自动创建项目（草稿）→ 进入项目详情页（编辑模式）

在详情页中（支持分段保存）：
  完善项目信息 → 添加标段包（轻量弹窗）→ 添加明细 → 
  完善标段包信息 → 添加邀请供应商（如适用）

点击确认 → 完整度校验 → 状态变为已确认 → 进入项目执行
```

  - 经评审的最高投标价法：本期不启用
- 报价方式业务含义：
  - 金额报价：总价包干
  - 费率报价：百分比报价
  - 单价报价：清单单价，按实际量结算
- 是否报名规则：
  - 询比、谈判、直接采购、竞价(邀请)：默认"不报名"或"报名需审核"，只读
  - 竞价(公开)：默认"报名需审核"，可选
  - 公开招标、邀请招标：不显示此字段

**功能路径**：`采购系统 → 项目管理 → 我的项目`

## 六、关键业务规则

### 6.1 建项规则

- 仅可选择状态为「已生效」或「已复核」的采购立项
- 选择采购立项后自动创建项目（草稿状态），项目名称自动带出
- 支持分段保存：用户可分多次完善项目信息、标段、明细、邀请供应商
- 标段创建时仅需填写标段名称，其余字段在详情面板中完善
- 点击「确认」时进行完整度校验，校验通过后状态变为已确认
- 至少添加一个标段方可确认
- 每个标段下至少添加一条明细方可确认
- 项目明细从关联的采购立项明细中选择，不可手动新增
- 标段与明细是一对多关系，一条明细只能关联一个标段
- 删除项目后，已关联的采购立项将被释放，可重新用于建项

### 6.2 采购方式特有规则

#### 公开招标/邀请招标
- 显示资审方式字段（公开招标显示，邀请招标不显示）
- 显示「是否两阶段项目」字段
- 不显示「满足开启家数」「是否报名」等字段

#### 询比采购（公开）/询比采购（邀请）
- 显示「询比方式」字段（公开/邀请，只读）
- 满足开启家数默认 3 家
- 是否报名默认「不报名」，只读
- 评审办法默认「经评审的最低投标价法」，可编辑
- 邀请版需在标段下添加多个供应商

#### 谈判采购（公开）/谈判采购（邀请）
- 显示「谈判方式」字段（公开/邀请，可编辑）
- 满足开启家数默认 2 家
- 是否报名默认「不报名」，只读
- 邀请版需在标段下添加多个供应商

#### 直接采购
- 显示「是否事前公示」字段（默认"是"，只读）
- 满足开启家数默认 1 家，只读
- 是否报名默认「报名需审核」，只读
- 评审办法默认「经评审的最低投标价法」，只读
- 每个标段只能关联 1 个供应商

#### 竞价（公开）/竞价（邀请）
- 显示「竞价方式」字段（公开/邀请，可编辑）
- 显示「竞价范围」字段（必填）
- 满足开启家数默认 3 家，只读
- 是否报名默认「报名需审核」，可编辑
- 显示竞价专属字段（竞争形式、竞争类型、延时方式、竞争时长、延时时长、起拍价、价格梯度等）
- 不显示概算价、评审办法、是否两阶段项目等字段
- 邀请版每个标段只能关联 1 个供应商

### 6.3 保存与确认

**分段保存**：编辑页面任何时刻均可保存当前内容，状态保持草稿。

**完整度校验**：点击「确认」时一次性校验以下必填项：
- 项目名称、项目概况已填写
- 采购方式对应的条件字段已填写（如询比方式、谈判方式、竞价方式等）
- 至少一个标段
- 每个标段必填项均已填写（详见各采购方式的标段信息条件字段）
- 每个标段下至少有一条明细
- 邀请类采购方式下，每个标段已添加邀请供应商

### 6.4 撤销与删除

- **撤销**：已确认状态可撤销回草稿，撤销后可继续编辑。不可撤销的条件待确认（如已发布公告/已开标等）。
- **删除**：草稿状态可删除项目。删除前弹出二次确认弹窗，确认后删除项目及其所有标段、明细、供应商数据。

## 附录：业务术语说明

### 评审办法
- **经评审的最低投标价法**：投标先通过全部实质性审查，再取评审总价最低者中标。适用于通用标准化项目。
- **综合评估法**：总分100分，分报价分、技术分、商务分加权计分。适用于技术复杂、定制化项目。
- **经评审的最高投标价法**：本期不启用。

### 报价方式
- **金额报价**：总价包干，一口价。
- **费率报价**：百分比报价，最终结算=审定造价×中标费率。
- **单价报价**：清单单价，按实际量结算。

### 联合体投标
两家及以上独立公司签联合协议，捆绑成一个投标主体。中标后一起履约、连带担责。

### 公开方式计算规则
- 公开：公开招标、询比采购(公开)、谈判采购(公开)、竞价(公开)
- 邀请：邀请招标、询比采购(邀请)、谈判采购(邀请)、直接采购、竞价(邀请)

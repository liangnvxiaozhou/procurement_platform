## 一、功能概述（重写）

功能入口位于租户列表操作列"平台使用费"按钮，点击后打开**独立页面**，按 项目类别（招标/非招）× 组织形式（自行采购/委托采购）形成 4 个 Tab，每个 Tab 展示该租户的收费规则列表，支持多条规则管理，同一时间仅 1 条生效。

```
租户管理 → 平台使用费
          │ 打开独立页面
          ▼
┌─ 平台收费标准 ───────────────────┐
│ [自采招标] [自采非招] [委托招标] [委托非招]  ← 4 个 Tab
│                                    │
│  规则列表（多条规则，1条生效）      │
│  支持：新增 / 编辑 / 生效 / 失效    │
└────────────────────────────────────┘
          │ 读取生效规则
          ▼
采购立项 → 标段包（读取租户当前生效的收费规则）
```

```
┌─ 平台收费标准 ──────────────────────────────────────────────────────────────┐

│  ══════════════════════════════════════════════════                          │
│                                                                             │
│  规则名称  [                ]   [ 搜索 ]  [ 重置 ]                          │
│                                                                             │
│  [ 新增规则 ]  [ 关闭 ]                                                     │
│                                                                             │
│  ┌──────────────────────────────────────────────────────────────────────┐   │
│  │规则编号    │规则名称│计算金额│是否计费│生效状态│创建时间│操作       │   │
│  │───────────┼───────┼───────┼───────┼───────┼───────┼────────────│   │
│  │ZB-ZX-001  │标准A  │预算价 │按阶梯 │● 生效 │2026-05│编辑/删除/失效│   │
│  │ZB-ZX-002  │简易B  │预算价 │按阶梯 │○ 无效 │2026-06│编辑/删除/生效│   │
│  │ZB-ZX-003  │免费版 │预算价 │不计费 │○ 无效 │2026-06│编辑/删除/生效│   │
│  └──────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

**Tab 默认行为**：进入页面默认显示第一个 Tab（自采招标），如某 Tab 无规则则显示空状态提示。

**操作列逻辑**：

| 操作 | 条件 | 说明 |
|------|------|------|
| 编辑 | 始终可用 | 打开规则详情抽屉 |
| 删除 | 是否生效 = 无效 | 已生效的规则不可删除（按钮置灰） |
| 生效 | 是否生效 = 无效 | 点击后同 Tab 其他规则自动失效，当前规则生效 |
| 失效 | 是否生效 = 生效 | 点击后当前规则变为无效（该 Tab 下无生效规则） |

### 3.3 规则详情抽屉（新建/编辑，右侧滑出）

```
┌─ 规则详情 ──────────────────────────────────────────────────┐
│                                                              │
│  ▾ 基本信息                                                  │
│                                                              │
│  * 规则名称      [                                        ] │

│         费率、单价项目，中标（成交人）平台使用费均摊。        │
│    项目类别      [ 招标        ] （Tab 锁定，只读）         │
│    组织形式      [ 自行采购    ] （Tab 锁定，只读）         │
│  * 是否计费      [ ● 按阶梯计费 / ○ 不计费 ]               │
│                                                              │
│  ▾ 计算规则（仅当"是否计费"= 按阶梯计费 时显示）          │
│                                                              │
│  区间为左闭右开 [最小值, 最大值)，最小值单位：万元；        │
│  收费金额单位：元                                            │
│                                                              │
│  [ 添加档位 ]                                                │
│                                                              │
│  ┌───────────────────────────────────────────────────────┐   │
│  │序号│最小值(万元)│最大值(万元)│收费金额(元)│操作     │   │
│  │────┼───────────┼───────────┼───────────┼───────────│   │
│  │ 1  │    0      │  100      │  2,000    │编辑/删除  │   │
│  │ 2  │  100      │  500      │  3,500    │编辑/删除  │   │
│  │ 3  │  500      │ 无上限 ☑  │  5,000    │编辑/删除  │   │
│  └───────────────────────────────────────────────────────┘   │
│                                                              │
│                                                              │
│                  [ 取消 ]  [ 保存 ]                          │
└──────────────────────────────────────────────────────────────┘
```

**"是否计费"联动逻辑**：
- 选择"按阶梯计费"：计算规则区域展开，可配置阶梯
- 选择"不计费"：计算规则区域折叠隐藏，不允许配置（已配置的阶梯数据保留但计费时被忽略）

### 3.4 添加/编辑档位弹窗
保持不变（同现 3.3）。

## 四、业务规则（重写）

将业务规则按职责分组重写，去掉原 R11/R12/R13，按新逻辑组织：

### 4.1 阶梯校验规则（R1-R7，保留不变）
- R1 最小值起始 / R2 连续性 / R3 区间有效 / R4 无上限唯一 / R5 金额正数 / R6 至少一档 / R7 收费递增提示

### 4.2 规则生命周期规则

| 编号 | 规则 | 说明 |
|------|------|------|
| R8 | 编号唯一 | 规则编号全局唯一，按 Tab 前缀+序号 区分 |
| R9 | 新建默认无效 | 新建规则默认为"无效"，需手动点击"生效" |
| R10 | 生效规则唯一 | 每 Tab 同一时间只能有 1 条规则生效 |
| R11 | 生效自动切换 | 点击规则 B"生效"时，同 Tab 原生效规则 A 自动变为无效，需二次确认 |
| R12 | 编辑保护 | 已生效的规则不可编辑（按钮置灰），需先失效或新建 |
| R13 | 删除保护 | 已生效的规则不可删除，需先设为无效 |
| R14 | 即时生效 | 切换生效规则后，新立项按新规则计费；已立项项目已快照不受影响 |

### 4.3 计费逻辑规则

| 编号 | 规则 | 说明 |
|------|------|------|
| R15 | Tab 匹配查询 | 项目立项时，根据 (项目类别, 组织形式) 定位对应 Tab 的生效规则 |
| R16 | 无生效规则 | 对应 Tab 无生效规则时，视为不计费（平台使用费 0 元） |
| R17 | 是否计费字段 | 规则"是否计费"=不计费时，无论阶梯内容，平台使用费为 0 元 |
| R18 | 代理服务费独立 | 代理服务费与平台使用费独立，在标段信息中单独配置 |

## 五、与其他功能的关系（更新）

### 5.1 项目立项（更新读取逻辑）

- 根据项目的 (项目类别, 组织形式) 定位到对应 Tab
- 读取该 Tab 下的生效规则
- 若生效规则"是否计费"=不计费，或无生效规则 → 平台使用费 0 元
- 若生效规则"是否计费"=按阶梯计费 → 按阶梯表计算

### 5.2 全局日志审计（增加生效切换记录）

- 规则的新建、编辑操作记录在全局操作日志
- **生效/失效切换**（含自动切换）必须记录在全局操作日志
